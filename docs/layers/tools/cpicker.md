## Description

This layer provides color picker for SpaceVim. It requires neovim 0.10+.

## Install

To use this configuration layer, add it to your `~/.SpaceVim.d/init.toml`.

```vim
[[layers]]
name = "tools#cpicker"
```

## Key bindings

| Key Binding | Description       |
| ----------- | ----------------- |
| `SPC i p c` | open color picker |
